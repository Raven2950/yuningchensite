我们说话时，常常忘记语言是有重量的。一句话说出口，它会下坠、会撞击、会在某个人心里泛起涟漪。本作品将这个隐喻变成了可交互的视觉体验：你在输入框中敲下任意文字，它们就会化作雨滴，带着质量和速度从天而降，落入水面，荡开一圈圈涟漪。

When we speak, we often forget that language has weight. A sentence, once uttered, falls—it strikes, it sends ripples through someone's mind. This work turns that metaphor into an interactive visual experience: type anything into the input field, and your words become raindrops, falling with mass and velocity from the sky, splashing into the water, sending ripples outward in widening circles.

概念｜Inspiration Concept · Inspiration

这是一个基于粒子系统的文字雨模拟。在粒子系统框架中，每个粒子拥有位置、速度、加速度和生命周期——本作品将这些物理属性赋予了文字本身。每一个字符不再是屏幕上静止的符号，而是一个受力运动的粒子：它有质量，会被重力拉扯，会在到达目标时消散并转化为另一种形态——涟漪。用户的输入行为成为粒子系统的"发射源"，打字这一日常动作被转化为一场持续的、可控的雨。

This is a text-rain simulation built on a particle system. Within the particle system framework, each particle possesses position, velocity, acceleration, and lifespan—this work grants those physical properties to text itself. Each character is no longer a static symbol on screen but a force-driven particle: it has mass, is pulled by gravity, and upon reaching its target dissolves into another form—a ripple. The user's typing becomes the particle system's emitter; the everyday act of typing is transformed into a continuous, controllable rain.

机制｜Mechanism Mechanism

系统由两类粒子构成：

The system consists of two particle types:

  • 文字雨滴（TextDrop）：每个雨滴是一个完整的粒子，携带位置、速度、加速度和质量。每帧受到恒定向下的重力（0.06），通过牛顿第二定律 a = F / m 计算加速度，质量越大的粒子下落越沉稳。速度经过 0.995 的阻尼衰减以保持视觉稳定。每个雨滴在生成时被分配一个随机的"目标高度"，到达后触发涟漪并重新从画面上方刷新，携带用户最新输入的文字重新降落。
  • 涟漪（Ripple）：雨滴触地时，根据撞击速度计算涟漪强度——速度越快，扩散越剧烈。涟漪以椭圆形向外扩展，透明度随生命值线性衰减直至消亡，模拟水面波纹的自然消散。
  • TextDrop: each raindrop is a full particle carrying position, velocity, acceleration, and mass. Every frame it receives constant downward gravity (0.06); acceleration is computed via Newton's second law a = F / m, so heavier particles fall more steadily. Velocity is damped by 0.995 each frame for visual stability. Each drop is assigned a random "target height" at spawn; upon reaching it, a ripple is triggered and the drop respawns from above carrying the latest input text.
  • Ripple: when a drop lands, ripple intensity is calculated from impact speed—faster means more dramatic spread. Ripples expand as ellipses with opacity decaying linearly until they vanish, simulating the natural dissipation of surface waves.

两类粒子通过"撞击事件"耦合：雨滴的死亡即涟漪的诞生，形成一个持续循环的粒子生态。

The two particle types are coupled through "impact events": a raindrop's death is a ripple's birth, forming a continuously cycling particle ecology.
